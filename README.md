# mario-mandel.github.io

Personal site for documenting technical learning projects and tutorials.

## Live Site

[mario-mandel.github.io](https://mario-mandel.github.io)

## What's Here

This is a simple static site showcasing:
- Technical tutorials and learning guides
- Project documentation
- Experiments and builds

Currently featuring a comprehensive Docker learning guide with hands-on exercises.

## Tech Stack

- Pure HTML/CSS (no frameworks)
- Hosted on GitHub Pages
- Mobile responsive
- Zero build process

## Structure

```
├── index.html           # Homepage
├── docker-week1.html    # Docker tutorial
└── README.md
```

## Local Development

Just open `index.html` in a browser. No build step needed.

## Adding Content

1. Create new HTML page following existing structure
2. Add link in `index.html` posts-grid section
3. Commit and push - GitHub Pages auto-deploys
